
# Restaurant Recommendation Chatbot + User Study

Group project for a Master's course Methods in AI Research at Utrecht University.

## Goal
Design and implement a conversational rule-based agent that recommends restaurants based on user
preferences (e.g., cuisine, budget, location), and evaluate its usability through a user study.

## My contribution
- Implemented the machine learning systems quantitative and qualitative evaluation 
- Retrieving user preferences from prompts
- Performed the data analysis in R

## Methods
- Conversational AI (rule-based)
- Recommendation logic based on user constraints
- User study design and analysis

## Tools
- Python
- Data analysis: R

## NOTE: 
This repository contains a cleaned and partial version of the original group project,
shared for portfolio and educational purposes. Full data can't be shared because of privacy of the users.
